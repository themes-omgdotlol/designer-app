# Profiler

Build it yourself:

- Windows
  Clone this repository
  run: npm install
  ru:n npm runbuild:win

- Mac
  Clone this repository
  run: npm install
  ru:n npm runbuild:mac

- Linux
  Clone this repository
  run: npm install
  ru:n npm runbuild:linux
